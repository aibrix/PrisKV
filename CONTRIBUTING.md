## Coding style
### C
clang-format-18 with [config file](.clang-format) is required, please run
`make format` before submitting codes!

A hint of LLVM [automatic installation script](https://apt.llvm.org/).

### Python
yapf==0.32.0 is required, please run `pip3 install yapf==0.32.0 && make format`
before submitting codes!

Thanks!
